À propos de cette application

Cette application utilise l’API OMDb (www.omdbapi.com) pour récupérer les informations sur les films.

Elle est développée en Angular avec :

  • Formulaires réactifs (ReactiveFormsModule)
  • Directive pour afficher les listes de films
  • Service HTTP pour communiquer avec l’API OMDb

Pour rechercher un film, rendez-vous dans l'onglet Films.
